Vision
======
Nutzer können auf einer Karte die Positionen ihrer Teammitglieder sehen und über die Karte kommunizieren (zum Beispiel
über Marker).

Wichtig - Server
================
- Nutzer müssen einen eigenen Namen vergeben können
- Geo-Position eines Clients kann auf dem Server gespeichert und abgerufen werden
- Mehrere Clients können in einer Gruppe untergebracht werden und sehen die Positionen ihrer Gruppenmitglieder
    - Clients können eine Gruppe erstellen
    - Clients können die Gruppe wieder verlassen
    - Clients können aus der Gruppe entfernt werden
    ? Wie werden Clients für eine Gruppe eingeladen?
- Die Positionen der Gruppenmitglieder werden in regelmäßigen Zeitabständen aktualisiert
- Gruppenmitglieder können Marker auf der Karte setzen und entfernen
    - ein Marker hat eine fixe Position und ein Icon

Wichtig - Client
================
- Karte und Position der Gruppenmitglieder werden angezeigt
- Positionen der Gruppenmitglieder werden aktualisiert
- Marker werden angezeigt

Optional
========
- Nutzer können ihr eigenes Icon wählen mit dem sie bei den anderen Nutzern angezeigt werden
- An Markern kann abgelesen werden, wer sie gesetzt hat und wann sie gesetzt wurden
- Nutzername ist je nach Gruppe änderbar
- Marker können bearbeitet werden (nicht nur setzen & löschen)
- Custom-Karten können verwendet werden
- Gruppen können benannt werden
- Gruppen können in Hierarchien zusammengefasst werden
