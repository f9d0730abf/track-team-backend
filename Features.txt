Individualisierung

- Farbe für Marker von Nutzer wählbar
- Marker kann von Nutzer mit eigenem Icon (rund) versehen werden
- Nutzer können eigene Namen für andere Nutzer vergeben

Styling

- Richtung, in die sich der Nutzer bewegt, wird angezeigt
- Äußerer Marker Kreis dreht sich
- Bei neuen Positionen bewegen sich die Marke dort hin

Kamera 

- Kamera kann auf alle Marker mittels Button Klick gerichtet werden
- Kamera kann mit Button Klick auf eigenen Marker gerichtet werden

Karte

- Wann das letzte Mal Positionen abgerufen wurden, wird als Text angezeigt
- Veraltete Positionen von Spielern (wo es länger kein Update gab) werden gesondert markiert
- Eigener Marker wird gesondert markiert
- Alte Marker verblassen und verschwinden
- Am Nutzer Marker kann man erfragen, zu welchem Nutzer der Marker gehört
- Es gibt eine Liste aller Marker, bei der, wenn man auf einen Maker klickt, die Kamera darauf gerichtet wird
- statt Google Maps soll OpenStreetMap verwendet werden können
- Karten für einen Bereich sollen heruntergeladen werden können (um vor Ort Bandbreite zu sparen)

Einstellungen

- Senden von eigener Position kann deaktiviert werden
- Position Abfrage Intervall kann angepasst werden
- Position senden Intervall kann angepasst werden
- Verbindungsdaten werden angezeigt (Server, Gruppe, Passwort)
- Intervall bis Marker verblassen kann allgemein angepasst werden
- Nutzer können ihre Position anonym senden (ohne Farbe, Icon und Name)

Custom Maps

- Eine eigene Karte kann vom Gruppenleiter hochgeladen werden
- Eine eigene Karte kann vom Gruppenleiter positioniert  werden
- Eine eigene Karte kann angezeigt werden
- Eine eigene, konfigurierte Karte kann von Gruppenmitgliedern heruntergeladen und verwendet werden

Eigene Marker

- Marker können gesetzt und von anderen geteilt werden
- Geteilte Marker können gelöscht werden
- Geteilte Marker können geändert werden
- Marker können auf einer beliebigen Position der Karte gesetzt werden
- Marker können an der Stelle an der man sich gerade befindet gesetzt werden
- Eigene Marker (Bild) kann vom Gruppenleiter hochgeladen werden
- Eigene Marker können von Gruppenmitgliedern heruntergeladen werden
- Gruppenmitglieder können geteilte Marke Bilder verwenden
- An einem Marker kann man sehen, wer und wann er gesetzt wurde
- Marker können auf permanenten oder verblassend gesetzte werden
- Intervall in dem Marker verblassen, kann am Marker gesetzt werden

Zeichnen auf Karte

- Linien können auf die Karte gezeichnet und geteilt werden
- Pfeile können auf die Karte gezeichnet und geteilt werden
- Beliebige Polygone können auf die Karte gezeichnet und geteilt werden

Verwaltung

- Nutzer können eine Gruppe verlassen 
- Nutzer können vom Gruppenleiter entfernt werden
- Gruppenleiter kann an einen anderen Nutzer übergeben werden
- Gruppenleiter kann Einstellung für Mitglieder vorgeben (Position Sende Intervall, Position Abfrage Intervall, ...)
- Gruppenleiter kann alle Gruppeneinstellungen, Marker und Overlays (alles, was nicht nutzerspezifisch ist) exportieren
- Gruppenleiter kann exportiere Gruppeneinstellungen wieder importieren

Sonstiges

- Fehlgeschlagene Position Sendung werden bei einer erfolgreichen Sendung mitgeschickt und daraus eine Abdeckungskarte erstellt
- Kommunikation zwischen Server und Client erfolgt verschlüsselt
- Gruppen können zu übergeordneten Gruppen zusammengefasst werden
- Man kann auch im Browser einer Gruppe beitreten, dann aber nur Marker sehen und keine eigene Position senden
